# Bienestar con Anel - Feel Great Website

A static HTML website for promoting wellness products from Unicity's Feel Great system.

## Overview

This website showcases the Feel Great system, featuring two main products: Unimate and Balance. It's designed to provide information about natural supplements and connect visitors with Anel for personalized wellness guidance.

## Features

- Responsive design
- Smooth scrolling navigation
- Interactive elements
- WhatsApp integration for contact
- Social media links
- Product information and ordering links

## Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- JavaScript (vanilla)

## Getting Started

Simply open `index.html` in a web browser to view the website.

## Customization

To customize this website:
1. Replace placeholder images with actual photos
2. Update contact information and links
3. Modify product descriptions as needed
4. Adjust colors and styling in the CSS
